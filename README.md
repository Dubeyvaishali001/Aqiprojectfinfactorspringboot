# Aqiprojectfinfactorspringboot


TO run this Repo

clone it 
git clone https://github.com/Dubeyvaishali001/Aqiprojectfinfactorspringboot.git
cd Aqiprojectfinfactorspringboot
cd aqi-backend

need to modify  .env file
AQICN_TOKEN=your_token_here   #create for your own from https://aqicn.org/api/
API_URL=https://api.waqi.info/feed
PORT=5000

Open a terminal to  Run backend:

mvn spring-boot:run
will run 
open another terminal parallely
npm run dev

http://localhost:5174/
